# Lumina — AI Knowledge Studio

Transform PDFs into interactive learning experiences powered by RAG.

## Features (planned)
- Chat with PDFs (RAG-powered)
- AI Avatar with voice interaction
- Auto-generated quizzes, flashcards, mind maps
- PDF-to-video animations
- Persistent memory across sessions

## Tech Stack
- **Backend:** FastAPI (Python 3.12)
- **Frontend:** Next.js 14 + TypeScript
- **Database:** Supabase (PostgreSQL)
- **Vector DB:** Qdrant Cloud
- **LLM:** Claude API
